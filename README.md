# Phone Input
Widget to enter phone numbers in international format

### Gradle
    
    repositories {
        maven{url "https://github.com/shaubert/maven-repo/raw/master/releases"}
    }
    dependencies {
        compile 'com.shaubert.ui.phone:library:1.0'
    }
